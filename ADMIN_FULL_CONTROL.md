# لوحة التحكم الموسعة بدون Firebase Storage

هذه النسخة تجعل لوحة المشرف تتحكم في:

- معاينة المنصة داخل لوحة التحكم.
- تغيير اسم المنصة، عنوان الترحيب، الوصف، الألوان، الخلفية، الشعار، صورة البانر، رابط واتساب، وشريط إعلان.
- إضافة/حذف الخدمات.
- إضافة/حذف الأعمال السابقة.
- إضافة/حذف الشهادات بروابط صور وملفات خارجية.
- إضافة/حذف المميزات.
- إضافة/حذف أقسام جديدة في الصفحة الرئيسية.
- إضافة/حذف نماذج طلب مخصصة.
- إضافة/حذف إشعارات تظهر في جرس الإشعارات.
- إضافة/حذف آراء العملاء.

## مهم
لا تستخدم هذه النسخة Firebase Storage. ضع روابط الصور والملفات فقط.

أمثلة مصادر مجانية للروابط:
- الصور: ImgBB أو Postimages أو Cloudinary Free.
- ملفات PDF: Google Drive أو Dropbox أو GitHub أو MediaFire.

## قواعد Firestore المقترحة للتجربة

```js
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
```
