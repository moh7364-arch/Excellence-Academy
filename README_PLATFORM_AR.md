# AcademiaHub - منصة أكاديمية شاملة

## أهم التعديلات
- أصبحت الصفحة الرئيسية عامة ولا تطلب تسجيل الدخول عند الفتح.
- تم نقل التسجيل إلى `auth.html` ويظهر فقط عند طلب خدمة.
- تمت إضافة قائمة جانبية كاملة للأقسام المطلوبة.
- تمت إضافة صفحات/أقسام تفصيلية لكل خدمة مع تخصصات وفروع وصور.
- تمت إضافة قسم النشر الدولي والمحلي مع بوابات الدول والمجلات المحلية النموذجية.
- تمت إضافة خبراء بأسماء وهمية قابلة للتعديل من لوحة المشرف.
- تمت إضافة أعمال سابقة وقوالب أصلية قابلة للتحميل.
- تمت إضافة شهادات نشر ومشاركة وخبرة وترجمة.
- تمت إضافة لوحة مشرف موسعة لإدارة الأقسام، الخدمات، الخبراء، المجلات، الأعمال، القوالب، الشهادات، النماذج، الإشعارات، الألوان والخلفيات.
- تم إلغاء الاعتماد على Firebase Storage؛ الصور والملفات تستخدم روابط خارجية فقط أو ملفات داخل مجلد assets.

## روابط مهمة
- الصفحة الرئيسية: `index.html`
- تسجيل/إنشاء حساب العميل: `auth.html`
- لوحة المشرف: `admin.html`

## ملاحظات قانونية
الملفات والقوالب داخل مجلد `assets/templates` و `assets/samples` أصلية مولدة خصيصاً للاستخدام داخل المنصة، وليست نسخاً من أعمال محمية.

## قواعد Firestore المقترحة بعد الاختبار
```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
```
