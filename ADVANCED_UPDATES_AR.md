# تحديثات التطوير المتقدم

تمت إضافة طبقة جديدة للمنصة تشمل:

## 1. نظام طلبات العملاء
- عند اختيار خدمة يظهر نموذج طلب ذكي.
- يتم حفظ الطلب في مجموعة Firestore باسم `orders`.
- يتم تجهيز رسالة واتساب وبريد إلكتروني بعد الحفظ.
- لا يحتاج Firebase Storage، لأن الملفات تُرسل كرابط Google Drive أو أي رابط خارجي.

## 2. لوحة العميل
ملف جديد: `customer.html`

تحتوي على:
- إجمالي الطلبات.
- الطلبات قيد التنفيذ.
- الطلبات المكتملة.
- قائمة الطلبات وحالة كل طلب.
- زر طلب جديد وزر خروج.

## 3. إدارة الطلبات من لوحة المشرف
تمت إضافة تبويب جديد في `admin.html` باسم: الطلبات.

يمكن للمشرف:
- مشاهدة جميع الطلبات.
- قراءة تفاصيل الطلب.
- فتح رابط ملف العميل.
- تغيير حالة الطلب إلى: جديد، قيد المراجعة، قيد التنفيذ، بانتظار العميل، مكتمل، ملغي.
- حذف الطلب.

## 4. ملفات JavaScript الجديدة
- `js/orders-firebase.js`: يحفظ الطلبات في Firestore.
- `js/customer.js`: يعرض طلبات العميل حسب بريده.

## 5. ملاحظات مهمة
إذا ظهرت رسالة صلاحيات، استخدم مؤقتاً قواعد Firestore التالية أثناء الاختبار:

```js
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
```

وبعد التأكد من التشغيل، يمكن تحسين قواعد الأمان لاحقاً.
